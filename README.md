# My-project
Project name: rock, scissor, paper game
team member: Kim Yeshu (22400125)

This project is about an extended rock, scissor, paper game.  
It aims to allow users to enter their names and play against the computer while collecting points based on wins and draws.  
The main goal of this project is to practice handling user input, scoring systems, and game logic.  
Through this project, we expect to improve our coding skills and understanding of simple game development.

## User's guide
Enter one of the following:
rock / paper / scissors
The computer will randomly choose one.
The result (win, lose, draw) will be displayed.

if You broke the highest score, You can choose write your score.
Enter one of the following:
yes / no

yes
